  • View every language

    Each translation unit becomes a row with one column per language. Pin a default and compare against a single target when you want focus.

  • Edit in place

    Click any cell to update its text. Empty cells let you add missing translations — a new <tuv> is created on save. Edits stay highlighted and can be filtered at a glance.

  • Manage languages

    Add a new column, rename a code, or remove a language entirely. Every code is matched with its English name automatically.

  • Search & replace

    Find a phrase across the whole file, then replace it in only the languages you pick — with optional case matching and a live match count before you commit.

  • Bulk delete

    Select any number of translation units with row checkboxes and remove them together. A confirmation step keeps destructive actions deliberate.

  • Export anytime

    Download a clean TMX with every edit, addition, and deletion applied. Original attributes, namespaces, and structure are preserved.
