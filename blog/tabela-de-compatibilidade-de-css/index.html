Tabela de compatibilidade de CSS

Existem dezenas de propriedades de CSS que podem ser utilizadas na concepção de um website, no entanto, muitas delas não funcionam em todos os browsers. Existe uma lista de compatibilidade das propriedades do CSS em vários browsers.

A tabela contém todas (ou quase todas) as propriedades do CSS e mostra a compatibilidade com 20 browsers diferentes. A bolinha vermelha significa que a propriedade não é suportada pelo browser. A bolinha amarela significa que a propriedade é suportada mas pode conter bugs. Finalmente, a bolinha verde significa que a propriedade é totalmente suportada pelo browser.

A tabela tem uma coisa muito boa: podemos clicar num dos browsers e ver a lista de compatibilidade das propriedades só para esse browser, ou então podemos clicar na propriedade e podemos ver a sua compatibilidade com os vários browsers. Muito bom!
