Portal das Curiosidades com mais novidades

This is an archive post from 2004–2015. Some images may no longer load.

Após a remodelação do layout efectuada no início do mês, e depois de duas semanas a resolver os problemas que iam surgindo com as mudanças, eis que ontem foi lançada mais uma nova funcionalidade que, penso eu, trará muitos mais visitantes ao Portal das Curiosidades.

A partir de ontem o Portal das Curiosidades passa também a ter uma secção de Jogos. Basicamente são pequenos jogos grátis em Flash que temos disponíveis para os membros registados (ainda não decidi se abra esta secção aos visitantes também).

O melhor de tudo é que as pontuações obtidas em cada jogo são gravadas e associadas ao utilizador registado no fórum, o que permite criar uma competição de "curiosos" para tentarem bater os recordes uns dos outros nos vários jogos :)

Tudo isto foi possível usando o SMF Arcade, um MOD do SMF (o sistema que está por detrás do Portal) que permite instalar jogos e integrar tudo com a base de dados existente.

"Let the games begin!"
