Conferência @media 2006

Por Ivo Gomes, em Quarta-feira, 15 de Fevereiro de 2006 | 7 Comentários

Nos dias 15 e 16 de Junho vai decorrer mais uma conferência @media.

A @media é a conferência de web design mais importante da Europa. Esta conferência junta vários dos mais conhecidos nomes do web design mundial para falarem sobre as últimas novidades, tendências, melhores práticas e técnicas mais avançadas do mundo do web design.

As apresentações vão englobar uma série de temas, desde a concepção de interfaces para utilizadores, web standards, CSS, DOM, acessibilidade e usabilidade.

Nunca fui a nenhuma destas conferências, mas este ano gostava de ir... Os oradores são bem conhecidos e já leio os seus blogs há algum tempo. São quase como referências ou influências para mim :)

Dos oradores presentes, destaco os seguintes:

  • Andy Budd - especialista em experiência do utilizador e em web standards;
  • Dan Cederholm - web designer, autor do popular SimpleBits e de dois livros muito bons sobre web design;
  • Tantek Çelik - trabalhou na Microsoft como representante do W3C e criou muitos dos hacks de CSS mais conhecidos para o Internet Explorer;
  • Andy Clarke - consultor e web designer apaixonado pelos web standards, autor do popular And All That Malarkey;
  • Dave Shea - autor do CSS ZenGarden e do popular blog Mezzoblue;
  • Veerle Pieters - designer gráfica que costuma publicar dicas de XHTML/CSS e de design gráfico;
  • Cameron Moll - conhecido web designer;
  • Eric Meyer - autor de um dos melhores livros para aprender a programar em CSS e de outros livros sobre web design;
  • John Hicks - designer web bastante conhecido pelo seu blog onde escreve sobre web standards e CSS. É o autor dos logótipos do Firefox e Thunderbird;
  • Rojer Johansson - profisisonal web com uma paixão pelos web standards, usabilidade e acessibilidade;
  • ...entre outros.

O programa parece-me excelente!
