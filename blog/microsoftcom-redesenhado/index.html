Voltar ao início

Microsoft.com redesenhado

Por Ivo Gomes, em Sexta-feira, 27 de Agosto de 2004 | 20 Comentários

Finalmente a Microsoft resolveu fazer a reconcepção do seu website segundo os Web Standards. Finalmente deixaram de usar tabelas para definir o layout (de 40 tabelas passaram para cerca de 7) e o tamanho das páginas caiu para apenas 11Kb!

Antigamente a Microsoft tinha duas versões diferentes do seu site, consoante o tipo de browser que os utilizadores tinham. Com este novo formato, o site é compatível com qualquer browser.

No entanto nem tudo são rosas... O código usado nas páginas ainda contém alguns erros ao nível da validação, mas é de louvar este pequeno esforço. Talvez a Microsoft esteja a mudar e adopte finalmente os Web Standards!
