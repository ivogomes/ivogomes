Toda a informação do Mundo

Literalmente! O Newsmap é uma aplicação que mostra numa janela do browser todas as notícias do Google News de uma forma hierarquizada e fácil de compreender. Este sistema de organização de informação chama-se "treemap" (mapa de árvore). Os "treemaps" mostram bastante informação num espaço muito pequeno. O que o Newsmap conseguiu fazer foi organizar essa informação por categorias de cores de forma a que seja mais fácil de ler.

Tal como diz o Javier Cañada (meu ex-colega na IconMedialab), a cor indica a secção, a intensidade cromática indica a actualizade da notícia e o tamanho indica a relevância.

Perfeito! Mas eu discordo num aspecto... é demasiada informação! Há coisas que são impossíveis de ler devido ao tamanho minúsculo das letras.

No entanto há que salientar o trabalho que foi desenvolvido no Newsmap por Dan Albitron e Marcos Weskamp uma vez que conseguiram organizar de uma forma bastante intuitiva toda a informação (o que não é fácil dado a enorme quantidade de informação disponível).

No site CyberGeography podem ver vários exemplos de mapas de informação, e como podem verificar, é difícil organizar tanta informação num pequeno ecrã.

Se quiserem podem também dar uma olhadela pela história dos "treemaps".

A Universidade de Maryland (EUA) está a desenvolver um software para criar "treemaps". O coordenador do projecto é o bem conhecido Ben Shneiderman. Em 1990, o Professor Shneiderman desenvolveu o primeiro "treemap" para demonstrar como as 14 pessoas no seu laboratório estavam a usar o disco rígido de um computador. Existe uma versão para download do software desenvolvido por Scheiderman na Universidade de Maryland.

Existe uma empresa que usa esta tecnologia aplicada à informação das empresas. The Hive Group é o nome da empresa e fornecem serviços para transformar a informação de uma base de dados num mapa hierárquico que pode ser visualizado em qualquer browser. Existe também uma versão de Demo do programa.
