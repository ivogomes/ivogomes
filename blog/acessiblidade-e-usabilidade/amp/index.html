Acessibilidade e Usabilidade

Ivo Gomes
15 anos ago

Para inaugurar este site, nada melhor do que fazer uma breve comparação entre Usabilidade e Acessiblidade.

Hoje em dia, toda a gente fala da Usabilidade, análises heurísticas, testes com utilizadores... Excelente, é sinal que finalmente as pessoas se começam a preocupar com o Utilizador!

No entanto, nem sempre um site com boa usabilidade tem uma boa Acessibilidade.

Um dos casos mais comuns de falta de acessibilidade são os Formulários. É com o uso de formulários que os utilizadores entram em contacto com os responsáveis pelo website, por isso convém que sejam, no mínimo, acessíveis a todos. Existem algumas tácticas para os tornar acessíveis e não é assim tão dificil como isso.

Podem ver alguns exemplos neste site que explica o que fazer e o que não fazer quando se constroi um formulário. Se quiserem testar os vossos "Forms" sugiro que usem o Bobby.

Depois temos o já conhecido texto "Alt" que deve acompanhar todas as imagens. Por falar em imagens, elas só devem aparecer caso façam parte do contexto em que estão inseridas. O uso de imagens no layout deve ser feito através de CSS (Cascading Style Sheets).

Não menos importante é uso de uma linguagem simples e de fácil compreensão. Como queremos que os utilizadores compreendam aquilo que lhes dizemos, se o fazemos de uma forma confusa?

Para quem quer saber como tornar o seu site mais acessível, aconselho uma visita ao acessibilidade.net e a aprender a conceber sites com o uso de CSS.

Só como curiosidade, façam o teste do Bobby aos vossos websites e olhem bem para os resultados! Podem fazer também um teste ao ivogomes.com se quiserem...
