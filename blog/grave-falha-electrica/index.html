Grave falha eléctrica!!

This is an archive post from 2004–2015. Some images may no longer load.

Devido a uma grave falha no sistema eléctrico do servidor em que este site está alojado, todos os conteúdos (texto, bases de dados, e-mails) foram irremediavelmente perdidos!

Bem, irremediavelmente porque eu não fiz nenhum backup...

Sendo assim tive que reenviar todos os conteúdos para o site e felizmente, graças à cache do Google consegui copiar todos os artigos que tinha perdido e espero que estejam cá todos.

Usando a cache do Google é possível ver uma página web que foi guardada nos servidores do Google. Isto permite ver uma versão antiga da página que fica guardada no Google. Graças a este sistema consegui ver todos os artigos que tinha publicado anteriormente e que já não existiam no meu site. Depois foi só fazer "copy & paste".

Obrigado Google!!!

PS: Infelizmente, todos os comentários dos artigos foram irremediavelmente perdidos.
