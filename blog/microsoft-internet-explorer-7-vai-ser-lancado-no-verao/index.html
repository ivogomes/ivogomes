Microsoft Internet Explorer 7 vai ser lançado no Verão

A Microsoft está a preparar-se para lançar uma nova versão do seu browser, o Internet Explorer. Segundo a Microsoft, o IE7 será lançado ainda antes do lançamento do novo sistema operativo Longhorn. Provavelmente o IE7 será distribuido no Service Pack 3 (mais uma actualização para o Windows XP) ou como um programa separado.

Os analistas dizem que o lançamento do novo IE antes do sistema operativo Longhorn se deve ao sucesso que o browser da Fundação Mozilla (o Firefox) tem tido junto dos cibernautas. Nos últimos meses o Firefox tem ganho muitos utilizadores que deixaram de usar o IE. Pela primeira vez desde o início da guerra dos browsers, o Internet Explorer passou a ter menos de 90% da quota de mercado, e os números continuam a descer.

Segundo Ross Rubin, analista do NPD Group:

"Se não existisse o Firefox, provavelmente teriamos que esperar pelo lançamento do Longhorn para ver uma actualização do Internet Explorer"

Esperemos que esta actualização tão esperada do IE venha corrigir todos os bugs existentes nas versões anteriores e que finalmente a Microsoft adopte os webstandards!
