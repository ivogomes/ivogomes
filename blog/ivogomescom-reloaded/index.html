ivogomes.com Reloaded!

This is an archive post from 2004–2015. Some images may no longer load.

Foi feita hoje uma renovação do site ivogomes.com. Foram introduzidas algumas novidades, umas estéticas, umas técnicas e outras de usabilidade.

Comecemos pelas estéticas:
Sempre gostei de ver sites com uma foto no topo. Umas maiores, outras menores, mas fica sempre bonito. Aqui9 vou colocar algumas fotos tiradas por mim.

As inovações técnicas:
Foi introduzido um novo menu, feito atravésde uma técnica diferente da anterior. O autor desta técnica é o Didier Hilhorst e podem ver como funciona, ou até mesmo copiar para vocês o código que ele criou para fazer um menu destes.

Finalmente, as inovações de usabilidade:
Existe agora uma opção para abrir os links numa nova janela. Quem quiser pode fazê-lo de forma fácil e rápida. Ao clicar na caixinha existente na barra lateral, todos os links externos abrirão numa nova janela. Os links internos (dentro do site www.ivogomes.com) continuarão a abrir na mesma janela, idependentemente de ter sido seleccionada ou não a opção de abrir os links numa nova janela.
