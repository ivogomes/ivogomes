Microsoft aposta nos CSS

This is an archive post from 2004–2015.

O novo site da MSN foi lançado hoje. A grande notícia é que o site foi totalmente concebido em XHTML 1.0 Strict e CSS. O esforço é de louvar, principalmente vindo da Microsoft que é bastante criticada por causa dos bugs existentes no seu browser Internet Explorer que fazem com que não suporte correctamente algumas propriedades de CSS.

O site foi renovando, no entanto está um pouco "quadrado", poderiam ter feito melhor...

A grande novidade da MSN é o novo motor de busca cujo objectivo é destronar o Google do primeiro lugar. Foram investidos vários milhões de dólares em campanhas publicitárias para que o MSN se torne no motor de busca mais utilizado pelos internautas (o que é muito difícil, dado a posição que o Google detém neste momento).

Via StopDesign.
