Influências

Por Ivo Gomes, em Terça-feira, 14 de Fevereiro de 2006 | 13 Comentários

Todos nós, quando construimos algo, temos influências do trabalho realizado por outras pessoas. É sempre bom ter algo para usar como referência, algo que consideramos ser bom e que queremos também atingir no nosso trabalho.

Eu tenho algumas referências e sinto-me influenciado por elas a vários níveis: webdesign, usabilidade, acessibilidade, etc... mas neste caso vou falar apenas do webdesign. Para a concepção desta versão do blog utilizei várias influências provenientes de outros blogs e sites que costumo visitar. Vou referenciar as três principais:

Mark Boulton

http://www.markboulton.co.uk/journal
Esta foi a principal referência a nível do layout global. Um layout minimalista, com poucas imagens e com as cores que uso actualmente. Foi a principal referência para fazer a mudança do antigo layout para o novo.

Hicks Design

http://www.hicksdesign.co.uk
Excelente uso do branco para preenchimento de espaços. Desta forma os conteúdos são apresentados de forma simples e sem distrações adicionais. O uso do espaço em branco torna o layout muito mais limpo e é mais fácil centrarmo-nos nos conteúdos.

Ordered List

http://www.orderedlist.com
Inspiração para o layout da homepage. Ao contrário dos blogs normais, este layout dá muito mais ênfase ao artigo mais recente e permite chamar a atenção para outros artigos que se perderam no meio dos arquivos. Além disso, a barra inferior permite incluir outros conteúdos para além do simples blog.

Inspirados...

Ultimamente, para grande surpresa minha, tenho recebido alguns e-mails a perguntarem-me se podem utilizar o meu layout como base para os seus sites. Outros ainda perguntam se posso disponibilizá-lo como um template para o WordPress. E finalmente, outros infelizes simplesmente copiam descaradamente o código e colocam o nome deles como autor...

Apesar de tudo, é bom saber que o trabalho que desenvolvi serve também de inspiração para outras pessoas :)
