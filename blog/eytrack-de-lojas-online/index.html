Usabilidade

Eytrack de lojas online

This is an archive post from 2004–2015. Some images may no longer load.

Já vos falei antes do Eyetrack, a técnica que permite saber para onde olham os utilizadores quando usam um computador. Esta técnica permite saber quais as zonas do ecrã que nos chamam mais a atenção.

Num dos meus artigos anteriores, foi feita uma análise ao Google, ou seja, para onde olham os utilizadores quando fazem uma pesquisa no popular motor de busca.

Neste caso foram testados vários sites de comércio electrónico para saber quais os locais que chamam mais a atenção dos utilizadores (apenas lhes foi pedido para vaguearem pela página inicial).

Um dos sites testados foi o da Amazon:

A linha branca indica o limite do ecrã com uma resolução de 1024x768 pixels.

Os resultados

Na página inicial os utilizadores conseguem encontrar rapidamente o tipo de produtos que querem pesquisar: "books", "electronics and photo", "music" e "dvd" são os itens mais visualizados no menu principal.

Uma das descobertas interessantes foi que, como é sabido, o canto superior esquero é o local esperado para os utilizadores encontrarem o nome ou logótipo do site que estão a visitar, sendo por isso uma zona de grande destaque e visibilidade do site. O que a Amazon fez aqui foi usar esse espaço para incluir um anúncio "FREE Trial" que fez com que mais de 50% dos utilizadores tivessem visto essa publicidade. Muito engenhoso, mas é uma forma de enganar os utilizadores para chamar a atenção para algo. Ao longo do site existem mais alguns exemplos deste tipo de situações.

Apesar destas alterações "manhosas" à forma como os utilizadores esperam que a informação esteja disponibilizada (indo contra algumas regras de usabilidade), a maior parte dos utilizadores não se sentiu aborrecido com isso.
O que a Amazon fez foi apenas mudar ligeiramente algumas coisas, fazendo com que os utilizadores explorem um pouco mais o site. No caso do canto superior esquerdo, o logótipo da Amazon está lá, só que está ligeiramente mais à direita do que o esperado. Desta forma, eles conseguiram cumprir as regras de usabilidade, mas com algumas diferenças que fazem com que os utilizadores vejam coisas que de outra forma provavelmente não veriam... Engenhoso!
