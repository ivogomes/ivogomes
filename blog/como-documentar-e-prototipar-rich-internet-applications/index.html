Como documentar e prototipar "Rich Internet Applications"?

This is an archive post from 2004–2015. Some images may no longer load.

A tecnologia AJAX e as "Rich Internet Applications (RIA's) revolucionaram a forma como os utilizadores interagem com os novos websites. Para quem desenha este tipo de websites, torna-se difícil documentar todas as funcionalidades que mudam consoante uma acção do utilizador, uma vez que numa mesma página, a informação pode ser disponibilizada de formas diferentes, dependendo do seu estado.

Experimentem tentar criar um protótipo estático de uma aplicação deste género e já ficam a saber a dificuldade que é.

No site UXMatters foi publicado um guia com informações bastante interessantes para podermos usar nos nossos wireframes. A solução é usar regiões destacadas e setas com informação sobre a opção seleccionada pelo utilizador:
