Microsoft recomenda uso de software Apple

This is an archive post from 2004–2015. Some images may no longer load.

A Microsoft está lentamente a acabar com algumas das suas aplicações para Mac.
O Internet Explorer/Mac deixou de ser desenvolvido em 2003 e no dia 31 de Janeiro deixará de ser possível fazer o seu download na própria página da Microsoft.
Em relação ao IE, além de ter muitos bugs (ainda mais do que nas versões para Windows), era um browser que já quase ninguém usava no Mac. A própria Microsoft recomenda a sua substituição para um browser mais recente tal como o Apple Safari!

It is recommended that Macintosh users migrate to more recent web browsing technologies such as Apple's Safari.

Esta semana recebi a notícia que o Windows Media Player está prestes a ser também terminado. A versão mais recente para Mac OS X é a 9 enquanto que para Windows penso que está prestes a sair a versão 11. Na página da Microsoft eles recomendam o uso de outro software que permite ver vídeos WMV na aplicação Apple Quicktime.

E hoje soube que eles também estão a equacionar acabar com o Office, mas dizem que não será nos próximos 5 anos! Em relação às duas aplicações anteriores (IE e WMV) não me preocupa muito que a Microsoft acabe com elas uma vez que há outros programas que desempenham melhor essas funções, mas em relação ao Office, ainda não encontrei nenhum que fosse tão rápido e funcional que o pudesse substituir... Já experimentei o NeoOffice mas não é lá grande coisa e é lento. O OpenOffice idem (ainda não saiu a versão 2.0 para Mac, mas sei que é muito mais rápida do que a versão anterior). Vamos aguardar...
