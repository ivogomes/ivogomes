WordPress recruta moderadores e participantes para testes de usabilidade

Por Ivo Gomes, em Segunda-feira, 4 de Maio de 2009 | 5 Comentários

A equipa do WordPress (o software de blogging que uso aqui no meu blog) está a recrutar participantes e moderadores voluntários para conduzirem testes de usabilidade ao interface de administração do WordPress. Os moderadores terão que ter já alguma experiência na condução de testes de usabilidade mas os participantes poderão ser qualquer pessoa que use o WordPress.

Neste momento ainda não existe uma página para que os participantes voluntários se possam registar, mas em breve será criada uma base de dados global onde todos os utilizadores se poderão registar para participar e ajudar a melhorar o interface do WordPress.

A última versão do WordPress (2.7) sofreu grandes alterações no interface com base nos resultados de vários estudos de usabilidade realizados ao longo do ano passado. Estas alterações foram muito bem recebidas pela maioria dos utilizadores uma vez que tornou a aplicação muito mais fácil de usar e o acesso a todas as páginas de administração ficou praticamente disponível em apenas um ou dois cliques.

Para a nova versão, a ideia é realizar testes de usabilidade com todos os elementos do interface, mesmo os mais pequenos, de modo a melhorar ainda mais a experiência de utilização. O facto de se estar a pedir voluntários prende-se com a necessidade de testar o interface com vários tipos de utilizadores em vários locais distintos no mundo (um utilizador americano fará com certeza uma utilização diferente do sistema do que um utilizador na China ou no Médio Oriente). Por ser impossível realizar estes testes de forma global, a equipa do WordPress optou por pedir ajuda à sua enorme comunidade para ajudar a melhorar este excelente produto.
