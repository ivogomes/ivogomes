Acessibilidade

Links em artigos impressos

This is an archive post from 2004–2015. Some images may no longer load.

Depois de ler o artigo "Improving Link Display For Print" na sempre interessante A List Apart, resolvi que tinha que aplicar a mesma técnica para melhorar a acessibilidade dos meus links sempre que algum utilizador imprima um artigo.

A ideia é simples e ajuda a resolver um problema actual. Se imprimirmos um artigo qualquer da Internet e este tiver algum link lá pelo meio, não nos é possível saber qual o endereço desse link sem ter que visitar novamente o site e procurar pelo artigo para poder clicar no respectivo link.


Com esta técnica, os links na página impressa passam a ter uma referência numérica que pode ser consultada no final do artigo. Desta forma permitimos que o utilizador que esteja a ler a página consiga identificar os links existentes na mesma sem ter que voltar ao site para poder clicá-los.


No final da página temos um rodapé com os endereços dos links existentes no artigo.

É claro que isto só funciona se tivermos uma folha de estilos definida para impressão, mas para isso recomendo a leitura de outro excelente artigo publicado na "A List Apart" e escrito pelo Eric Meyer: "CSS Design: Going To Print"
