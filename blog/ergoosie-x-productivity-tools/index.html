Pausas e Micro-Pausas II

Por Ivo Gomes, em Terça-feira, 14 de Junho de 2005 | 3 Comentários

Há algum tempo falei de um programa que ajudava as pessoas que trabalham com computadores a fazerem algumas pausas durante o dia para descansar a vista e os músculos.
Na altura recomendei que instalassem o WorkRave nos vossos computadores para ajudar a minimizar os efeitos de estar um dia inteiro a olhar para um ecrã de computador.
O problema é que continua a não haver uma versão para Mac deste software. Até agora não tinha encontrado nada semelhante para os utilizadores deste sistema operativo... até aparecer o Ergoosie-X!

O funcionamento do Ergoosie-X é bastante semelhante com o do WorkRave. O objectivo deste software é criar uma rotina no trabalho em que o utilizador se aperceba da forma como efectua o seu trabalho e se habitue a fazer pausas e micro-pausas regulares durante o dia. A diferença é que o WorkRave é grátis enquanto que o Ergoosie-X é pago.

Download: Ergoosie-X Demo (Mac OS X)
Download: WorkRave (Windows e Linux)

Actualização: Para Mac existe também o AntiRSI que faz basicamente a mesma coisa que o Ergoosie e o WorkRave, excepto os exercícios. Este software é grátis e é o que estou a usar agora :)

Encontrei ainda mais duas aplicações para Mac, mas não são grátis:

  • WorkPace
  • RestTimerX
