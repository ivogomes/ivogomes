Suporte para PCs Portáteis

This is an archive post from 2004–2015. Some images may no longer load.


A empresa Ergo Touch é especializada na criação de soluções para melhorar a interacão entre homens e computadores. Uma das suas mais famosas invenções é uma pequena plataforma desdobrável (Laptop Desk) onde podemos colocar o computador portátil e que nos fornece um espaço extra para colocar um rato ou um bloco de notas, por exemplo.

A plataforma também pode ser usada para elevar a posição do computador ou para o apoiar em cima dos braços de uma cadeira. É sem dúvida uma invenção interessante para quem usa o portátil no colo e não tem espaço para usar o rato ou para colocar outros objectos que são necessários em determinadas tarefas com o computador. Além disto, o Laptop Desk não é condutor de calor, pelo que evita o aquecimento das pernas quando se tem o portátil ao colo.

Olá morgi! Penso que finalmente resolvi a questão dos links externos. Removi a “caixinha” e inseri em todos os links externos uma imagem de um “globo”, que significa que o link seguirá para um site externo a este. Automaticamente, todos os links com o “globo” abrirão numa nova janela.

Penso que esta é a melhor solução.

PS: no XHTML o atributo “target” é inválido, por isso os links numa nova janela são abertos através de javascript.
