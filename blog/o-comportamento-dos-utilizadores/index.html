O Comportamento dos Utilizadores

Por Ivo Gomes, em Quinta-feira, 22 de Julho de 2004 | 0 Comentários


Normalmente, para identificar o comportamento dos utilizadores num determinado site, usamos testes de usabilidade com utilizadores reais, pedimos-lhes que executem uma determinada tarefa, e registamos o que eles fazem até conseguirem completar a tarefa com êxito. Isto pode ser um processo muito demorado (ter que registar tudo o que o utilizador faz até chegar ao seu objectivo). Há quem use várias técnicas: uma videocâmara apontada para o monitor; papel e lápis para registar por onde o utilizador passou e o que verbalizou; gravadores áudio; software de captura de movimentos do rato; etc...

Às vezes isto pode fazer com que os testes de usabilidade se tornem mais caros do que seria esperado. No entanto já há software que é capaz de "ler" os logs de acesso dos sites e transformar essas informações em dados que podem ser utilizados para efectuar uma análise de usabilidade. Com o ClickTracks Analyser é possível saber quantas pessoas clicaram em determinado link, de onde vieram, para onde foram, quanto tempo demoraram, etc... Isto tudo com gráficos excelentes e com percentagens por cima de cada link existente no site. É ainda possível fazer cruzamento de tabelas para tornar a análise ainda mais completa.

Podemos também criar grupos de utilizadores com base nos dados registados nos logs. Desta forma podemos fazer uma análise de usabilidade diferente para cada grupo de utilizadores.

Para sites de e-commerce, o ClickTracks Analyser fornece dados interessantes tais como indicar o número de pessoas que visitaram um determinado artigo, quantas clicaram para comprar, quantas compraram na realidade o artigo, se não compraram também podemos saber onde é que sairam do site (pode dar-nos uma ideia do local onde os utilizadores sentem dificuldade em continuar a compra), etc...

Finalmente, para os sites que fazem publicidade online é também possível saber de onde vêm os utilizadores e saber se as campanhas de publicidade estão a ser bem sucedidas ou não.

É a tecnologia a evoluir para facilitar o trabalho de quem tem de analisar o comportamento dos utilizadores!
