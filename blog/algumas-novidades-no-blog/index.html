Algumas novidades no blog

Por Ivo Gomes, em Segunda-feira, 7 de Novembro de 2005 | 14 Comentários

Este fim-de-semana aproveitei para testar algumas coisas novas aqui no blog.
Mudei o meu sistema de gestão de fotos antigo e passei a usar a API do Flickr juntamente com o WordPress para gerir as minhas fotos.
Mudei também o sistema de arquivo do blog e passei a usar um novo sistema que usa tecnologia AJAX. Finalmente, adicionei uma pré-visualização automática dos comentários ao formulário de comentários aos artigos.

Novo sistema de gestão de fotos (Flickr)

Como tenho pouco espaço aqui no servidor (cerca de 130MB), não posso colocar muitas fotos online. Ultimamente já estava a receber e-mails de alerta a dizer que já estava a usar 90% da capacidade máxima do disco e como tal, era preciso fazer alguma coisa.

A solução passava por criar uma conta num sistema externo de gestão de fotos e depois importá-las para o meu site através de uma API.
Primeiro experimentei o novo sistema de Fotos do Sapo que me foi recomendado pelo Pedro (pertencente à equipa do Sapo que desenvolveu o sistema). Criei uma conta e andei a experimentar um pouco. Depois resolvi procurar uma maneira de importar as fotos do Sapo através de uma API para o meu site, mas não encontrei nada no site que me ajudasse a fazer isso.

Resolvi então procurar uma solução mais simples, o Flickr. O Flickr é o maior site de gestão de fotos do mundo e foi recentemente comprado pelo Yahoo.
Já tinha conhecimento que eles tinham uma API que podia ser usada para ir buscar as fotos e disponibilizá-las da forma que quisermos e onde quisermos, mas como não sou programador, tive que procurar algo que já estivesse desenvolvido. Uma vez que o Flickr é usado por milhões de pessoas, é natural que hajam vários scripts e plugins desenvolvidos para tirar melhor partido de todas as funcionalidades disponíveis. E assim encontrei o FAlbum, um plugin para o WordPress que permite ir buscar as minhas fotos ao Flickr e disponibilizá-las aqui no site com o meu layout. Perfeito.

O único problema é que as contas grátis no Flickr só permitem a criação de 3 álbuns e têm um limite mensal de 20Mb de transferência de fotos. Então resolvi comprar uma conta no Flickr que me iria expandir o limite mensal para 2Gb e a possibilidade de ter fotos e álbuns infinitos. O custo de uma anuidade não é muito caro (21€) e aproveitei o sucesso dos anúncios do Google no Portal das Curiosidades para me pagarem a anuidade :)

O sistema está a funcionar lindamente apesar de ser um pouco mais lento do que se tivesse as fotos alojadas no meu servidor, mas em termos de gestão e organização das fotos, não há nada como o Flickr.

www.ivogomes.com/fotos

Novo arquivo (AJAX)

Juntamente com o novo sistema de gestão de fotos, instalei também um novo sistema de arquivo. Mais uma vez usei um plugin do WordPress para implementar este novo sistema.
A ideia é simples, ter um menu navegável por datas em que não é necessário carregar a página sempre que se selecciona um ano ou um mês. Para isso muito contribui o uso de tecnologia AJAX!
