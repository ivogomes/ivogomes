SPAM nos comentários

This is an archive post from 2004–2015. Some images may no longer load.

Ultimamente tenho sido alvo de ataques massivos de SPAM nos comentários. Todos os dias recebo perto de 100 comentários com links para sites de casinos e poker.

Felizmente o WordPress tem uma função para filtrar os comentários considerados SPAM e remete-os todos para aprovação. O problema é que todos os dias tenho que clicar em cento e tal comentários e dizer que não os aprovo. Já para não falar dos e-mails de notificação que recebo de cada vez que é detectado um comentário de SPAM e é enviado para a lista de aprovação.

A partir de hoje isso não voltará a acontecer. Instalei um plugin do WordPress que coloca uma imagem com um código de segurança que deve ser copiado para validar o comentário. Desta forma acaba-se o SPAM porque os comentários têm que ser validados pela própria pessoa. É um pouco chato estar a pedir ao utilizador que insira um código para validar o comentário, mas dado o estado actual da Internet em que tudo vale para enviar SPAM, há que tomar medidas drásticas.

Para quem usa o WordPress e esteja interessado em instalar o plugin:
