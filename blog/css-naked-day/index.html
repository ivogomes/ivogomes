CSS Naked Day

This is an archive post from 2004–2015.

Pergunta: Porque é que hoje o blog está tão feio?
Resposta: O site está a ser apresentado sem folhas de estilo, ou seja, nu! Hoje, dia 5 de Abril é o CSS Naked Day, e em todo o mundo, milhares de sites e blogs estão a participar para mostrar que é possível navegar num site olhando apenas para os seus conteúdos.

As folhas de estilo (CSS) são apenas uma maneira de dar uma apresentação (layout) aos conteúdos, mas se o texto estiver bem escrito usando a semântica e o código HTML estiver correcto, podemos dispensar os CSS e apresentar o site a nu. Desta forma, os estilos serão aplicados pelo próprio browser. Esta é também a forma como um leitor de ecrã (usado por pessoas que com deficiência visual) “vê�? o site e lê os seus conteúdos.

Desta forma, o site ivogomes.com estará sem folhas de estilo durante o dia 5 de Abril. O engraçado é que o site vai estar sem CSS durante 48 horas, ou seja, são as 48 horas durante as quais é dia 5 de Abril nalguma parte do globo. Para a malta portuguesa, isto representa o período entre as 12h do dia 4 e as 12h do dia 6 de Abril — mais hora, menos hora… hora…

Actualização:

O primeiro CSS Naked Day terminou e o site voltou a adoptar o seu layout habitual. para o ano há mais
